How to Fork the GitHub repository

  1. Create a GitHub account
  2. Create your own "fork" of the readordie repository by going to the repository and clicking the Fork button.


    3. Location of the Fork button on a different repo. It'll be in the same spot.

  3. Go to your new fork of the repo and clone that repository onto your computer with the Clone or download button.

  4. Use Terminal to cd into your newly cloned repo and enter git status, which should tell you you are up-to-date! Now you are ready to add a post.
